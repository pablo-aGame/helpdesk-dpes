HelpDesk DPES - Paquete multipágina (prototipo)
------------------------------------------------
Contenido:
  index.html        - Inicio de sesión
  registro.html     - Registro de usuario
  menu.html         - Menú principal
  ticket.html       - Formulario para levantar ticket (simulado)
  listado.html      - Listado con 2 tickets ficticios
  detalle.html      - Detalle del ticket "Requiere instalación de Office"
  ayuda.html        - Centro de asistencia
  styles.css        - Hoja de estilos (puedes editar colores/fuentes)

Cómo abrir localmente (sin internet ni servidor):
  1. Descomprime la carpeta 'helpdesk-dpes.zip'.
  2. Haz doble clic en 'index.html' para abrirlo en el navegador.
  3. Navega entre las páginas con los botones (Entrar, Levantar ticket, Listado, Centro de ayuda).
  4. No se requieren instalaciones; todo es HTML/CSS local.

Cómo subir a GitHub Pages:
  1. Crea un repositorio público en GitHub (ej: helpdesk-dpes).
  2. Sube todos los archivos y carpetas del paquete (index.html, registro.html, menu.html, ticket.html, listado.html, detalle.html, ayuda.html, styles.css).
  3. En GitHub, ve a Settings -> Pages -> Source y selecciona 'Branch: main' y carpeta '/ (root)'.
  4. Guarda. En unos minutos, tu sitio será visible en:
     https://TU_USUARIO.github.io/NOMBRE_REPO/index.html
  5. Nota: los archivos usan enlaces relativos, por lo que funcionarán igual en GitHub Pages.

Descripción de cada página:
  - index.html: Pantalla de inicio de sesión (simulada).
  - registro.html: Formulario de registro de usuario (simulado).
  - menu.html: Menú principal con accesos a módulos.
  - ticket.html: Formulario para crear un ticket (simulado, redirige al listado).
  - listado.html: Muestra 2 tickets ficticios (Falla en impresora - Cerrado, Requiere instalación de Office - Abierto).
  - detalle.html: Muestra información ampliada del ticket "Requiere instalación de Office".
  - ayuda.html: Centro de ayuda con acciones simuladas.
